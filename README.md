# 35L-StudyScheduler

Welcome to **StudyScheduler**! Our app is designed for students to easily create, find, and join online study groups with people that have similar coursework and schedules. 

To run our app, do the following:<br>
   - git clone https://github.com/ArchitWa/35L-StudyScheduler
   - cd server<br>
   - npm install<br>
   - npm run dev<br><br>
   - (open a new terminal)<br>
   - cd client<br>
   - npm install<br>
   - npm run dev<br>

Then, you should be able to see our homepage and press the "Log In" button at the top-right corner to get started! <br><br>


Some notable features:
 - User profiles include contact information, coursework, and schedule
 - Profile page includes recommended study groups based on similarity in coursework and schedules
 - Users can browse through study groups and query based on coursework and schedules
 - Study groups can be created by group leaders and users can send requests to join; group leaders can decide to accept/reject

This app was designed and implemented as our final project for Computer Science 35L at UCLA under Professor Paul Eggert during Winter 2026.
